formBuilder 2.0

June 9, 2016

It's been a while since there was any info on the long hyped formBuilder 2.0 so I thought I take a moment to address that.

The Bad News

There is no formBuilder 2.0

The Good News

Because its been renamed Formeo.

So why the re-branding?

  1. There are at least 3 other form building plugins or modules named formBuilder or similar. This made registering formBuilder with the same name across package managers difficult. Formeo is catchy and available on our favorite package managers.
  2. The second reason addresses the most common question asked in Gitter and GitHub issues- "How do I save form data to a database?". FormBuilder was intended to be a front-end tool only, leaving the backend problems like saving form data for engineers to solve. Formeo will still be available as a free stand alone front-end module but takes things further with formeo.io a service that will handle the data and provide analytics for completed forms.

Formeo is officially available for testing here and feedback is very much needed. This an alpha release so expect bugs and please feel free to contribute by reporting them here. With your help Formeo will be one of- if not the best JavaScript form building module on the internet.

Get Started

At a minimum formBuilder needs form-builder.min.js and form-builder.css, but here is a full page example. 
<!DOCTYPE html>
<html>
<head>
<title>Example formBuilder</title>
<link rel="stylesheet" type="text/css" media="screen" href="http://formbuilder.online/assets/css/form-builder.min.css">
</head>
<body>
  <textarea id="fb-template"></textarea>
  <script src="https://cdnjs.cloudflare.com/ajax/libs/jquery/2.1.4/jquery.min.js"></script>
  <script src="https://cdnjs.cloudflare.com/ajax/libs/jqueryui/1.11.4/jquery-ui.min.js"></script>
  <script src="http://formbuilder.online/assets/js/form-builder.min.js"></script>
  <script>
  jQuery(document).ready(function($) {
    'use strict';
    $(document.getElementById('fb-template')).formBuilder();
  });
  </script>
</body>
</html>
