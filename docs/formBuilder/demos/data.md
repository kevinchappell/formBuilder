# Getting Data

Get by Instance
```javascript
const formBuilder = $fbEditor.formBuilder();
const formData = formBuilder.formData; // getter
```
or
```javascript
const dataType = 'json' // optional 'js'|'json'|'xml', defaults 'js'
$fbEditor.formBuilder('getData', dataType)
```
