# Actions -> clearFields
Remove all fields from the editor with this handy action.

## Usage
```javascript
var formBuilder = $(fbEditor).formBuilder().data('formBuilder');
document.getElementById('clear-all-fields').onclick = function() {
    formBuilder.actions.clearFields();
};
```
