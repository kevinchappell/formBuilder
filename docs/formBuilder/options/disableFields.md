# disableFields
`disableFields` allows you to define and array of fields that should *not* be made available to users.

## Usage
```javascript
var options = {
      disableFields: ['autocomplete']
    };
$(container).formBuilder(options);
```

### List of built-in controls that can be disabled

- autocomplete
- button
- checkbox-group
- date
- file
- header
- hidden
- number
- paragraph
- radio-group
- select
- starRating
- text
- textarea
