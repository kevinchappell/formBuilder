# i18n
Introduced in v2.0, the `i18n` option allows you to define language specific settings.

## Usage
```javascript
var options = {
  i18n: {
    locale: 'ru-RU'
    //location: 'http://languagefile.url/directory/'
    //extension: '.ext'
    //override: {
    //    'en-US': {...}
    //}
  }
};
$(container).formBuilder(options);
```
