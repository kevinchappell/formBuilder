# onSave
Callbacks to run on editor save.

## Usage
```javascript
var options = {
      onSave: function(formData) {
          toggleEdit();
          $('.render-wrap').formRender({formData});
          window.sessionStorage.setItem('formData', JSON.stringify(formData));
        },
    };
$(container).formBuilder(options);
```
